## 管理者画面：大会管理タブ詳細

### タブ1：選手管理
```
[1日目] [2日目]  ← 日程切替
[1組] [2組] [＋組を追加]  ← 組切替

入力テーブル（6行固定）
┌──┬──────┬────────┬────────┬────────┬────┬────┐
│番│会員番号│氏名    │審判フラグ│所属    │クラス│削除│
├──┼──────┼────────┼────────┼────────┼────┼────┤
│1 │12345 │田中    │ □⚑    │大阪    │ A  │ ✕ │
└──┴──────┴────────┴────────┴────────┴────┴────┘
                              [この組を保存]

[選手組替]  ← 左端、ドラッグ&ドロップで組替可能

登録済み選手一覧（空きスロットはゴールド左ボーダー＋「－（空き）」表示）
```

### タブ2：点数登録
```
[1日目(R1-4)] [2日目(R5-8)] [↺ 再読込]
[全組] [1組] [2組]  ← グループフィルター

点数テーブル
┌──────┬────┬────┬────┬────┬────┐
│氏名  │ R1 │ R2 │ R3 │ R4 │小計│
├──────┼────┼────┼────┼────┼────┤
│田中  │[25]│[20]│    │    │ 45 │
└──────┴────┴────┴────┴────┴────┘

※セルをクリック → テンキーポップアップ表示
※25点セルは赤字表示（≥23で赤）
※「決定」で次の選手へ縦移動
```

### タブ3：成績確認
```
順位表（member_code でスコアと紐付け）
クラス別フィルター可能
```

### タブ4：設定
```
・大会情報編集（名前/射撃場/日付/セット番号/種目）
・QRコード表示（管理者用・閲覧者用、URLコピーボタン付き）
・危険ゾーン：メンバー・点数リセット（PW: "repros"）
```

## QRコードのURL

| QR種別 | リンク先 |
|--------|---------|
| 管理者用QR | `https://clay-shooting.vercel.app/admin/[id]` |
| 閲覧者用QR | `https://clay-shooting.vercel.app/viewer` |

> 閲覧者QRは特定大会ではなく大会一覧ページを指す（選手が自分で選ぶ）
