## 環境変数

### Vercelに設定が必要な環境変数

| 変数名 | 説明 |
|--------|------|
| `DATABASE_URL` | Neon PostgreSQL接続文字列 |

> ⚠️ 実際の値は `C:/projects/clay-shooting/開発メモ.txt` を参照

### ローカル開発用

`.env.local` ファイルをプロジェクトルートに作成：
```
DATABASE_URL=（開発メモ.txtの値を貼り付け）
```

---

## ローカル開発環境セットアップ

### 前提条件
- Node.js 18以上
- npm

### 手順
```bash
# 1. リポジトリクローン
git clone https://github.com/matsuo-mitsuhiro/clay-shooting.git
cd clay-shooting

# 2. 依存パッケージインストール
npm install

# 3. 環境変数設定
# .env.local を作成して DATABASE_URL を設定

# 4. 開発サーバー起動
npm run dev
# → http://localhost:3000 でアクセス可能
```

## デプロイ手順

### 通常のデプロイ（コード変更時）
```bash

```

- https://vercel.com のダッシュボードで確認

## DBセットアップ（初回のみ）

Neonダッシュボードの SQLエディタ で `db/schema.sql` を実行：
1. https://neon.tech にログイン
2. プロジェクト「Clay-shooting」を選択
3. 「SQL Editor」を開く
4. `db/schema.sql` の内容を貼り付けて実行
