# このマニュアルについて

クレー射撃大会 成績管理システムの操作マニュアルです。

## 対象読者

| 読者 | 内容 |
|------|------|
| 選手・閲覧者 | 大会成績の閲覧方法 |
| 大会管理者 | 大会の作成・運営・成績管理 |
| システム管理者 | マスターデータの管理・システム全体の設定 |

## システムのURL

| 画面 | URL |
|------|-----|
| トップページ | https://clay-shooting.vercel.app/ |
| 管理者ログイン | https://clay-shooting.vercel.app/admin |
| 閲覧者ページ | https://clay-shooting.vercel.app/viewer |

## ログイン方法

### 大会管理者
1. 管理者ログイン画面にアクセスします
2. 「会員番号」と「パスワード」を入力します
3. 「ログイン」ボタンをクリックします

### システム管理者
1. 管理者ログイン画面の下部にある「システム管理者の方はこちら」をクリックします
2. 「Googleアカウントでログイン」ボタンが表示されます
3. ボタンをクリックし、許可されたGoogleアカウントでログインします

### 選手・閲覧者
ログインは不要です。大会のQRコードを読み取ると閲覧ページにアクセスできます。閲覧時に「所属協会」と「氏名」を入力して「成績を確認する →」ボタンをクリックします。
