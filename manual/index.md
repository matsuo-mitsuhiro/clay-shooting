## このマニュアルについて

本マニュアルは、クレー射撃 成績管理システムの操作方法を説明したものです。

### 用語説明

| 用語 | 説明 |
|------|------|
| 大会管理者 | 大会の運営を担当するスタッフ。成績入力・選手管理を行う。 |
| システム管理者 | システム全体を管理する担当者。協会・射撃場・管理者アカウントを管理する。 |
| 閲覧者 | QRコードから大会ページにアクセスして成績を確認する選手・関係者。 |
| 種目 | トラップまたはスキート。 |
| ラウンド | 1ラウンドあたり最大25点。1日目はR1〜R4、2日目はR5〜R8。 |
| クラス | 選手の区分（AA・A・B・C）。 |

### アクセス方法

- **大会ページ（閲覧者）**: 大会会場に掲示されたQRコードを読み取ってアクセスします。
- **管理者ページ**: 大会管理者向けのQRコードを読み取るか、直接URLを入力してアクセスします。ログインが必要です。
