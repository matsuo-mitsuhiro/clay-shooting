# Q&Aを管理する

利用者からの質問の受付・回答と、Q&Aページへの掲載を管理します。

## 質問・Q&A管理画面

管理画面トップの「質問・Q&A管理」ボタンをクリックして開きます。

## タブ構成

画面上部に2つのタブがあります。

| タブ | 説明 |
|------|------|
| 質問一覧 | 利用者から届いた質問の一覧・回答 |
| Q&A管理 | Q&Aページに掲載する内容の管理 |

## 質問一覧タブ

### フィルター

質問のステータスで絞り込みができます。

- 「すべて」— 全件表示
- 「未回答」— 未回答の質問のみ
- 「回答済み」— 回答済みの質問のみ

### 質問一覧テーブル

| 項目 | 説明 |
|------|------|
| 受付日時 | 質問が届いた日時 |
| 会員番号 | 質問者の会員番号 |
| 氏名 | 質問者の氏名 |
| 所属協会 | 質問者の所属協会 |
| 状態 | 「未回答」または「回答済み」 |
| 操作 | 「詳細・回答」ボタン |

### 質問への回答

1. 「詳細・回答」ボタンをクリックします
2. 質問詳細モーダルが開きます
3. 質問者の情報（会員番号・氏名・所属協会・メール）と質問内容が表示されます
4. 「回答」欄に回答を入力します
5. 「回答を送信」ボタンをクリックします

回答済みの質問は、回答欄に「（回答済み・上書き可）」と表示され、内容を修正して再送信できます。

### Q&Aへの掲載

回答済みの質問をQ&Aページに掲載するには：

1. 質問詳細モーダルで「📋 Q&Aに掲載」ボタンをクリックします
2. 掲載フォームが表示されます

| 項目 | 必須 | 説明 |
|------|:---:|------|
| カテゴリ | | 「閲覧者の機能」または「運営管理者の機能」 |
| タイトル | * | Q&Aのタイトル（例: スコアの確認方法について） |
| 質問文（編集可） | | 質問内容（掲載前に編集可能） |
| 回答文（編集可） | | 回答内容（掲載前に編集可能） |

3. 「掲載する」ボタンをクリックします

## Q&A管理タブ

掲載済みのQ&Aが一覧で表示されます。

### Q&A一覧テーブル

| 項目 | 説明 |
|------|------|
| カテゴリ | 分類カテゴリ |
| タイトル | Q&Aのタイトル |
| 掲載日 | 掲載した日付 |
| 操作 | 「編集」「削除」ボタン |

### Q&Aの編集

「編集」ボタンをクリックすると編集モーダルが開きます。カテゴリ・タイトル・質問文・回答文を修正して「保存」ボタンで保存します。

### Q&Aの削除

「削除」ボタンをクリックすると確認ダイアログが表示されます。削除するとQ&Aページから非表示になります。
