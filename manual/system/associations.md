# 協会マスターを管理する

各協会の射撃場・中止連絡方法・注意書きを管理します。

## 協会マスター画面

管理画面トップの「協会マスター」ボタンをクリックして開きます。

## 画面構成

左側に協会一覧、右側に選択した協会の詳細が表示されます。

## 協会の選択

左側の一覧から協会名をクリックすると、右側に詳細が表示されます。

## 編集項目

| 項目 | 説明 |
|------|------|
| 所属射撃場 | 協会に紐づく射撃場をチェックボックスで選択 |
| 中止のお知らせ方法（400文字以内） | 大会中止時の連絡方法（例: 当協会のホームページにて公表いたします。） |
| 注意書き（400文字以内） | 申込フォームに表示される注意事項 |

## 保存

編集が完了したら「保存」ボタンをクリックします。

::: tip
ここで設定した「中止のお知らせ方法」と「注意書き」は、その協会が主催する大会の申込設定にデフォルト値として反映されます。
:::
