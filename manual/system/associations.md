# 協会マスターを管理する

クレー射撃の協会情報を管理します。大会の主催団体として使用されます。

## 協会マスター画面を開く

管理トップページの「協会マスター」をクリックします。

## 協会の追加

1. 「＋ 新規追加」ボタンをクリックします。
2. 以下の情報を入力します。

| 項目 | 説明 |
|------|------|
| 協会名 | 協会の正式名称を入力します。 |
| 射撃場 | この協会が管理する射撃場を選択します（複数選択可）。 |
| 中止連絡方法 | 大会中止時の連絡方法を入力します。閲覧者ページに表示されます。 |
| 注意書き | 大会に関する注意事項を入力します。閲覧者ページに表示されます。 |

3. 「保存」ボタンをクリックします。

## 協会の編集

一覧から協会名をクリック（スマートフォンの場合はタップ）すると編集画面が開きます。

変更後、「保存」ボタンをクリックします。

## 射撃場との紐付け

協会に射撃場を紐付けることで、大会作成時に選択できる射撃場が絞り込まれます。

- 大会管理者は、自分の協会に紐付けられた射撃場のみ選択できます。
- システム管理者はすべての射撃場を選択できます。
