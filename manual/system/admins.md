# 管理者アカウントを管理する

運営管理者のアカウントの作成・編集・削除を行います。この機能はシステム管理者のみ利用可能です。

## 運営管理者マスター画面

管理画面トップの「運営管理者マスター」ボタンをクリックして開きます。

## 管理者一覧

登録済みの運営管理者が一覧で表示されます。

| 項目 | 説明 |
|------|------|
| 会員番号 | 管理者の会員番号 |
| 氏名 | 管理者の氏名 |
| メール | メールアドレス |
| 現在の所属協会 | 選手マスターの所属に基づく現在の所属 |
| 状態 | 「有効」または「無効」 |
| 操作 | 「編集」「削除」ボタン |

### 所属協会で絞り込み

一覧の上部にある「所属協会で絞り込み：」ドロップダウンで、特定の協会の管理者だけを表示できます。

## 運営管理者を追加

1. 「＋ 運営管理者を追加」ボタンをクリックします
2. フォームに以下を入力します

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 会員番号 | * | 選手マスターに登録済みの会員番号（例: 12345） |
| 氏名 | * | 管理者の氏名（例: 山田 太郎） |
| メールアドレス | * | 連絡先メールアドレス |
| 初期パスワード | * | 8〜32文字、英字＋数字の組み合わせ |

3. 「作成」ボタンをクリックして登録します

## 招待QRコードの発行

「📱 招待QRコードを発行」ボタンをクリックすると、運営管理者が自分でアカウントを登録できるQRコードが発行されます。

::: warning 注意
使用期限24時間、利用者は1名に限定します。複数名を登録する場合は人数分、QRコードを発行してください。
:::

「URLをコピー」ボタンで招待URLをコピーできます。

## 管理者の編集

一覧の「編集」ボタンをクリックすると編集モーダルが開きます。

| 項目 | 説明 |
|------|------|
| メールアドレス | メールアドレスを変更 |
| 新しいパスワード | 変更する場合のみ入力（8〜32文字、英字+数字）。変更しない場合は空欄 |
| 有効 | チェックを外すとログイン不可になります |

「保存」ボタンで変更を保存します。

## 管理者の削除

一覧の「削除」ボタンをクリックすると確認ダイアログが表示されます。削除すると、その管理者はログインできなくなります。
