# 管理者アカウントを管理する

大会管理者のアカウント（ログインID・パスワード）を管理します。

## 管理者一覧画面を開く

管理トップページの「大会管理者マスター」をクリックします。

## 管理者の追加

1. 「＋ 新規追加」ボタンをクリックします。
2. 以下の情報を入力します。

| 項目 | 説明 |
|------|------|
| ログインID | メールアドレス形式で入力します。ログイン時に使用します。 |
| 氏名 | 管理者の氏名を入力します。 |
| 役職 | 管理者の役職を入力します。 |
| 所属（協会） | 管理する協会を選択します。この協会の大会のみ管理できます。 |
| パスワード | ログイン用パスワードを設定します。 |

3. 「保存」ボタンをクリックします。

::: tip
大会管理者は、自分が所属する協会の大会のみ管理できます。すべての大会を管理できるのはシステム管理者のみです。
:::

## パスワードの変更

1. 変更する管理者の「編集」ボタンをクリックします。
2. 新しいパスワードを入力します。
3. 「保存」ボタンをクリックします。

## 管理者の削除

1. 削除する管理者の「削除」ボタンをクリックします。
2. 確認ダイアログで「OK」をクリックします。

::: warning
削除した管理者はログインできなくなります。
:::
