# 成績を見る

大会の成績を閲覧する方法を説明します。

## アクセス方法

運営管理者から提供されたQRコードをスマートフォンで読み取ります。または、閲覧者ページのURLに直接アクセスします。

## ログイン

1. 「所属協会」をドロップダウンから選択します
2. 「氏名」を入力します
3. 「成績を確認する →」をクリックします

::: tip
「所属協会と氏名を保存する」にチェックを入れると、次回アクセス時に入力が省略されます。
:::

## 成績一覧の見方

ログイン後、大会の成績一覧が表示されます。

### 表示項目

| 項目 | 説明 |
|------|------|
| 順位 | 成績順位（上位3名はゴールド表示） |
| 氏名 | 選手名（審判は ⚑ マーク付き） |
| 組 | 所属する組番号 |
| 所属協会 | 所属する協会名 |
| クラス | AA / A / B / C |
| R1〜R4 | 1日目の各ラウンド点数（23点以上は赤字） |
| 1日計 | 1日目の合計点 |
| R5〜R8 | 2日目の各ラウンド点数（2日開催の場合のみ） |
| 2日計 | 2日目の合計点（2日開催の場合のみ） |
| 合計 | 全ラウンドの合計点 |
| 平均 | 1ラウンドあたりの平均点 |

### フィルター機能

成績一覧の上部に絞り込み機能があります。

- **クラス:** 「全て」またはクラス名（AA / A / B / C）をクリックして絞り込み
- **所属協会:** ドロップダウンから協会を選択して絞り込み

### 統計情報

フィルターの下に以下の統計が表示されます。

| 項目 | 説明 |
|------|------|
| 全体平均 | 表示中の全選手の平均点 |
| 上位6名平均 | 上位6名の平均点 |
| 表示件数 | 現在表示されている選手数 |

### ハイライト機能

自分の行をタップするとハイライト（色付き）表示されます。もう一度タップすると解除されます。ログイン時の氏名に一致する行は自動的にハイライトされます。

## 「↺ 更新」ボタン

画面右上の「↺ 更新」ボタンをクリックすると、最新の成績データに更新されます。

## 種目の表示

画面上部に大会の種目が「トラップ」または「スキート」と表示されます。
