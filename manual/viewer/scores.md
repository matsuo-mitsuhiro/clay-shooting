# 成績を見る

大会会場に掲示されたQRコードを読み取ると、大会の成績ページにアクセスできます。

## ログイン

成績ページを開くと、最初にログイン画面が表示されます。

- **氏名**: 自分の氏名を入力します。
- **所属**: 自分の所属団体を入力します。

入力後、「成績を見る」ボタンをタップしてください。

::: tip
一度ログインすると、ページを更新しても再度ログインする必要はありません。
:::

## 成績の見方

ログイン後、成績一覧が表示されます。

### 絞り込み

画面上部のフィルターを使って、表示する成績を絞り込めます。

| フィルター | 説明 |
|-----------|------|
| クラス | AA・A・B・C でフィルター |
| 所属 | 所属団体でフィルター |
| 氏名検索 | 氏名の一部で検索 |

### 成績表の見方

| 列 | 説明 |
|----|------|
| 組 | 射撃の組番号 |
| 番 | 組内の番号 |
| 氏名 | 選手氏名（⚑マークは審判） |
| クラス | 選手のクラス（AA・A・B・C） |
| 所属 | 所属団体 |
| R1〜R4 | 1日目の各ラウンドの点数 |
| 1日計 | 1日目の合計点 |
| R5〜R8 | 2日目の各ラウンドの点数（2日間開催の場合） |
| 2日計 | 2日目の合計点 |
| 合計 | 全ラウンドの合計点 |

::: info
**23点以上**のラウンドは赤字で表示されます。
:::

### タブ切り替え

1日間開催の大会は「1日目」タブのみ表示されます。
2日間開催の大会は「1日目」「2日目」タブを切り替えて確認できます。
