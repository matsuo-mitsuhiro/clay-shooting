# 申込を管理する

大会への申込者を管理します。申込者が申込フォームから登録すると、申込管理タブに一覧が表示されます。

## 申込管理タブを開く

大会詳細画面の「申込管理」タブをクリックします。

## 申込者の確認

申込一覧には以下の情報が表示されます。

| 列 | 説明 |
|----|------|
| 氏名 | 申込者の氏名 |
| 所属 | 申込者の所属団体 |
| クラス | 申込者のクラス（AA・A・B・C） |
| 会員番号 | 申込者の会員番号 |
| 申込日時 | 申込が行われた日時 |

## 申込者を手動で追加する

申込フォームを経由せず、管理者が直接申込者を登録することもできます。

1. 氏名・所属・クラス・会員番号を入力します。
2. 「申込管理リストに追加」ボタンをクリックします。

## 申込受付の期間設定

申込受付の開始日時・終了日時を設定できます。設定した期間外はフォームからの申込ができなくなります。

::: tip
申込受付を終了したら、「選手タブ」で出場選手を確定させてください。
:::
