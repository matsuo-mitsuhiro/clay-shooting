# 申込を管理する

大会への申込の受付・管理・選手管理への移行を行います。

## 申込設定

大会詳細画面の「申込設定」タブで、募集条件を設定します。

### 入力項目

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 募集人数 | * | 1日あたりの上限人数（例: 60） |
| 募集開始日時 | * | 申込受付の開始日時 |
| 募集終了日時 | * | 申込受付の終了日時 |
| キャンセル可能日時 | * | 申込者がキャンセルできる期限 |
| 射撃場開門時間 | * | 射撃場の開門時間 |
| 受付開始時間 | * | 受付の開始時間 |
| テストクレー放出時間 | * | テストクレーの放出時間 |
| 競技開始時間 | * | 競技の開始時間 |
| 中止のお知らせ方法（400文字以内） | * | 中止時の連絡方法 |
| 注意書き（400文字以内） | | 申込フォームに表示する注意事項 |

::: tip
射撃場開門時間を選択すると、受付・クレー・競技開始が自動設定されます（後から修正可）。
:::

設定が完了したら「申込設定を保存」ボタンをクリックします。

## 申込管理リスト

大会詳細画面の「申込管理」タブで、申込者の一覧を確認・管理します。

### 手動追加エリア

Web申込以外に、管理者が手動で選手を追加できます。

1. 「会員番号」を入力するとDBから氏名・所属協会・クラスが自動補完されます
2. 氏名のみ入力した場合は、氏名＋所属協会で検索されます
3. 参加日程（1日目 / 2日目 / 両方）を選択します
4. 「保存」ボタンをクリックして登録します

### 申込一覧

画面下部に申込管理リストが表示されます。

#### 表示項目

| 項目 | 説明 |
|------|------|
| 登録元 | 「Web」または「手動」のバッジ |
| 会員番号 | 選手の会員番号 |
| 氏名 | 選手名（クリックで編集可能） |
| 所属協会 | 所属する協会名（クリックで編集可能） |
| クラス | AA / A / B / C（クリックで編集可能） |
| 審判 | 審判フラグ（チェックボックスで切替可能） |
| 参加 | 参加日程（1日目 / 2日目 / 両方） |
| 申込日時 | 申込した日時 |
| ステータス | 申込中 / 移行済 / キャンセル |
| 操作 | キャンセル・削除・未移行に戻す |

#### フィルター

一覧の上部にフィルターがあります。

- **クラス:** 「全て」/ AA / A / B / C をクリックして絞り込み
- **所属協会:** ドロップダウンから選択して絞り込み

#### 操作ボタン

| ステータス | 操作 |
|-----------|------|
| 申込中（未移行） | 「キャンセル」 |
| 移行済 | 「キャンセル」 |
| 手動登録 | 「キャンセル」「削除」 |
| キャンセル済 | 「未移行に戻す」 |

::: warning 注意
選手管理に登録されている選手をキャンセル・削除する場合、「○日目 ○組○番に登録されている選手です」と警告が表示されます。
:::

### 選手管理に移行

申込者を選手管理に登録するには、「未移行 ○名 → 選手管理に移行」ボタンをクリックします。未移行かつ有効な申込者が自動的に選手管理の空きスロットに割り当てられます。

### 申込状況の表示

一覧の上部に、募集人数・申込人数・残り枠が日程別に表示されます。募集人数を超えた場合は赤字で「○名オーバー」と表示されます。

### 「最新に更新」ボタン

一覧データを最新の状態に更新します。
