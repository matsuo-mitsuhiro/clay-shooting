# 大会を作成する

新しい大会を作成する手順を説明します。

## 大会一覧画面

管理者ログイン後、大会一覧画面が表示されます。この画面から大会の作成・管理を行います。

## 新規大会の作成

1. 画面右上の「＋ 新規大会作成」ボタンをクリックします
2. 作成フォームが表示されます

### 入力項目

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 主催 | | ドロップダウンから協会を選択 |
| 大会名 | * | 大会の名称（例: 第○回○○大会） |
| 射撃場名 | | 主催に紐づく射撃場がドロップダウンに表示されます |
| 種目 | | 「トラップ」または「スキート」を選択 |
| 1日目 | | カレンダーから日付を選択 |
| 2日目（任意） | | 2日開催の場合のみカレンダーから日付を選択 |

3. 入力が完了したら「作成する」ボタンをクリックします
4. 大会一覧に新しい大会が追加されます

::: warning 注意
- 大会名は必須です。未入力の場合はエラーが表示されます
- 2日目の日付は1日目より後の日付を設定してください
- 1日目と2日目が連続していない場合は確認メッセージが表示されます
:::

## 大会の管理

大会一覧に表示された大会カードの「管理する →」ボタンをクリックすると、大会の詳細管理画面に移動します。

詳細管理画面には以下のタブがあります。

| タブ | 説明 |
|------|------|
| 大会設定 | 大会情報の編集・QRコード確認・危険ゾーン |
| 申込設定 | 募集人数・日時・スケジュールの設定 |
| 申込管理 | 申込者の一覧・手動追加・移行 |
| 選手管理 | 選手の組分け・編集・削除 |
| 点数登録 | 各ラウンドの点数入力 |
| 成績確認 | 成績一覧・順位設定 |
