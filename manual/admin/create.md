# 大会を作成する

管理者ページにログイン後、「＋ 新規大会作成」ボタンから新しい大会を作成します。

## 新規大会作成フォーム

「＋ 新規大会作成」ボタンをクリックすると、入力フォームが表示されます。

| 項目 | 必須 | 説明 |
|------|------|------|
| 主催 | - | 大会を主催する協会を選択します。 |
| 大会名 | ✅ | 大会の名称を入力します。（例：第○回○○大会） |
| 射撃場名 | - | 開催射撃場を選択します。 |
| 種目 | - | トラップまたはスキートを選択します。 |
| 1日目日付 | - | 大会初日の日付を選択します。 |
| 2日目日付 | - | 2日間開催の場合のみ入力します。1日開催の場合は空欄のままにしてください。 |

::: warning 日付について
- 2日目は1日目より後の日付のみ設定できます。
- 1日目と2日目が連続していない場合は確認メッセージが表示されます。
:::

入力後、「作成する」ボタンをクリックしてください。

## 大会作成後の流れ

大会を作成したら、大会詳細画面で以下の設定を行います。

1. **大会設定タブ** — QRコードを確認・共有する
2. **申込管理タブ** — 申込者を登録する（申込受付がある場合）
3. **選手タブ** — 出場選手を登録する
4. **成績タブ** — 点数を入力する

## 大会設定を変更する

大会作成後に内容を変更するには、大会詳細画面の「大会設定」タブを開きます。

- 大会名・射撃場・日付・種目を変更できます。
- **セット番号**（例：1番セット）を設定できます。成績表のヘッダーに表示されます。
- 変更後は「保存」ボタンをクリックしてください。

## QRコードの確認・共有

大会設定タブの「QRコード確認」セクションに3種類のQRコードがあります。

| QRコード | 用途 |
|---------|------|
| 管理者用QR | 大会管理者が管理画面にアクセスするためのQRコード |
| 閲覧者用QR | 選手・関係者が成績を閲覧するためのQRコード |
| 申込用QR | 大会への申込フォームにアクセスするためのQRコード |

「URLをコピー」ボタンでURLをコピーして、印刷・配布用の資料に貼り付けることができます。
