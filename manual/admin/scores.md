# 成績を入力する

各ラウンドの点数を入力します。

## 点数登録画面

大会詳細画面の「点数登録」タブを開きます。

## 日目の切替（2日開催のみ）

2日開催の場合、画面上部に「1日目 (R1-4)」「2日目 (R5-8)」のボタンが表示されます。クリックして日目を切り替えます。

## 組の選択

「全組」ボタンで全組を一覧表示できます。各組ボタン（1組、2組、3組...）で特定の組だけを表示できます。

## 点数テーブル

各組のテーブルが表示されます。

### 表示項目

| 項目 | 説明 |
|------|------|
| 氏名 | 番号付きで表示（例: 1. 西奥 直樹）。審判は ⚑ マーク付き |
| 所属協会 | 所属する協会名 |
| R1〜R4（1日目） | 各ラウンドの点数入力欄 |
| R5〜R8（2日目） | 各ラウンドの点数入力欄 |
| 小計 | 当日の合計点（自動計算） |
| CB | カウントバック値（1〜6） |
| FR | ファイナルシュートオフ値（1〜99） |

## 点数の入力方法

1. 入力したいセルをタップ（クリック）します
2. テンキーポップアップが表示されます
3. 数字を入力します（0〜25）
4. 「決定」をタップすると、入力が確定し**次の選手の同じラウンドに自動移動**します
5. 組の最後の選手で「決定」をタップするとポップアップが閉じます

::: tip
- 「C」ボタンで入力をクリアできます
- 23点以上は赤字で表示されます
- テンキーの外側（暗い部分）をタップするとポップアップを閉じます
:::

### CB・FRの入力

- **CB（カウントバック）:** セルをタップすると専用テンキー（1〜6）が表示されます
- **FR（ファイナルシュートオフ）:** セルをタップすると専用テンキー（1〜99）が表示されます

## 保存

各組のテーブル右上にある「この組を保存」ボタンをクリックして保存します。

::: warning 注意
- 会員番号が未登録の選手は点数を保存できません（「(会員番号なし)」と表示されます）
- 保存は組単位です。別の組に移動する前に保存してください
:::

## 「↺ 再読込」ボタン

画面左上の「↺ 再読込」ボタンで最新データを再読み込みできます。
