# 成績を入力する

選手の点数を組ごとに入力します。

## 成績タブを開く

大会詳細画面の「成績」タブをクリックします。

## 日目・組の選択

1. タブで「1日目」または「2日目」を選択します。（2日間開催の場合）
2. 組タブで入力する組を選択します。

## 点数の入力

### テンキーポップアップ

点数を入力するセルをクリックすると、テンキーポップアップが表示されます。

- 選手名と現在の点数が表示されます。
- **0〜22点**：青色で表示
- **23〜25点**：赤色で表示（高得点）
- 「C」ボタン：入力をクリアします。
- 「決定」ボタン：確定して次の選手（縦方向）に移動します。
- 「×」ボタン：キャンセルします（値は変更されません）。

::: warning
1ラウンドの入力可能範囲は **0〜25点** です。26以上はテンキーで入力できません。
:::

### 入力の流れ

同じラウンドを縦方向（選手順）に入力していく設計になっています。

1. R1の1番の選手をクリック
2. 点数を入力して「決定」
3. 自動的にR1の2番の選手に移動
4. 全選手のR1入力が終わったら、R2に移動して繰り返す

## 点数の保存

点数を入力したら「この組を保存」ボタンをクリックして保存します。

::: warning
保存前にページを移動すると、入力した点数が失われます。こまめに保存してください。
:::

### 保存時のバリデーション

保存時に0〜25の範囲外の値が含まれている場合、エラーメッセージが表示されます。

例：「1組、田中、R4 は現在 26 です。0〜25 以内の数字を入力してください」

エラーが表示されたら、該当の点数を修正してから再度保存してください。
