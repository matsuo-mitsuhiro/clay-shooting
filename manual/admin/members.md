# 選手を登録する

大会に参加する選手の組分け・編集・削除を行います。

## 選手管理画面

大会詳細画面の「選手管理」タブを開きます。

## 日目の切替（2日開催のみ）

2日開催の場合、画面上部に「1日目」「2日目」のボタンが表示されます。クリックして日目を切り替えます。

「1日目からコピー」ボタンをクリックすると、1日目の選手データを2日目にコピーできます。

## 組の選択

画面上部に組ボタンが表示されます（例: 1組、2組、3組...）。クリックして組を切り替えます。各組ボタンには登録人数が表示されます（例: 「1組（6名）」）。

「＋組を追加」ボタンをクリックすると、新しい組が追加されます。

## 選手一覧テーブル

選択した組の選手が表示されます。各組は最大6名（1番〜6番）で構成されます。選手が未登録のスロットは「－（空き）」と表示されます。

### 表示項目

| 項目 | 説明 |
|------|------|
| 番号 | 組内の番号（1〜6） |
| 会員番号 | 選手の会員番号 |
| 氏名 | 選手名 |
| 所属協会 | 所属する協会名 |
| クラス | AA / A / B / C |
| 審判 | 審判フラグ（⚑ マーク） |
| 成績 | 有効 / 失格 / 棄権（ドロップダウンで変更可能） |
| 操作 | 「削除」ボタン |

## 編集モード

テーブル右上の「編集」ボタンをクリックすると編集モードに入ります。

編集モードでは以下の項目を変更できます。

| 項目 | 編集方法 |
|------|---------|
| 所属協会 | ドロップダウンから選択 |
| クラス | ドロップダウンから選択（AA / A / B / C） |
| 審判 | チェックボックスで切替 |

変更後、「保存」ボタンをクリックして保存します。「キャンセル」ボタンで変更を破棄します。

## 成績の変更

「成績」列のドロップダウンから「有効」「失格」「棄権」を選択すると、即座に反映されます。失格・棄権の選手は成績確認で順位なし・最下位として表示されます。

## 選手の削除

「削除」ボタンをクリックすると確認ダイアログが表示されます。点数データがある場合は「○○の点数データも削除されます。削除しますか？」と警告されます。

## 選手組替

テーブルの下にある「選手組替」ボタンをクリックすると、組替モードに入ります。

組替モードでは、⠿ アイコンをドラッグして選手の位置を入れ替えることができます。全組の選手が一覧で表示されます。

| 項目 | 説明 |
|------|------|
| 組 | 組番号 |
| 番 | 組内の番号 |
| 氏名 | 選手名 |
| 所属協会 | 所属する協会名 |
| クラス | クラス |

並び替えが完了したら「組替を保存」ボタンをクリックします。「組替をキャンセル」で変更を破棄します。

## 申込済み未登録の警告

申込管理で申込済みだが選手管理に未登録の選手がいる場合、画面上部に警告バナーが表示されます。

> ⚠️ 申込済みだが選手未登録の方が ○名います

該当する選手の会員番号・氏名・参加日程が表示されます。
