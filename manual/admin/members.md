# 選手を登録する

大会に出場する選手を組・番号に割り当てて登録します。

## 選手タブを開く

大会詳細画面の「選手」タブをクリックします。

## 選手の登録

### 1日目・2日目の切り替え

タブで「1日目」「2日目」を切り替えて、それぞれの日程の選手を登録します。

::: tip
「1日目からコピー」ボタンを使うと、1日目と同じ組み合わせを2日目にコピーできます。
:::

### 選手情報の入力

各行に以下の情報を入力します。

| 項目 | 説明 |
|------|------|
| 会員番号 | 半角数字で入力します。全角は自動的に半角に変換されます。 |
| 氏名 | 選手の氏名を入力します。 |
| 所属 | 所属団体を入力します。 |
| クラス | AA・A・B・C から選択します。 |
| 審判 | 審判を担当する選手にチェックを入れます。成績表に ⚑ マークが表示されます。 |

### 組の追加

「＋ 組を追加」ボタンで組を追加できます。1組あたり最大6名まで登録できます。

### 保存

入力後、「この組を保存」ボタンをクリックして保存します。**組ごとに保存が必要です。**

## 選手の削除

登録済み一覧の「削除」ボタンで選手を削除できます。

- 点数データがない場合：確認なしで削除されます。
- 点数データがある場合：「点数データも削除されます」という確認メッセージが表示されます。

## 選手の組替え

「選手組替」ボタンをクリックすると、ドラッグ＆ドロップで選手の組・番号を変更できます。

1. 「選手組替」ボタンをクリックして組替えモードに入ります。
2. 選手をドラッグして別の組・番号にドロップします。異なる組への移動も可能です。
3. 「組替を保存」ボタンをクリックして保存します。
