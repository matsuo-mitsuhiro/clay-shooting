# 結果を公開する

成績確認と順位の設定を行います。閲覧者ページには自動的に反映されます。

## 成績確認画面

大会詳細画面の「成績確認」タブを開きます。

## 成績一覧

全選手の成績が順位順に表示されます。

### 表示項目

| 項目 | 説明 |
|------|------|
| 順位 | 成績順位（上位3名はゴールド表示） |
| 氏名 | 選手名（失格は「失格」、棄権は「棄権」のラベル付き。審判は ⚑ マーク） |
| 組 | 所属する組番号 |
| 所属協会 | 所属する協会名 |
| クラス | AA / A / B / C |
| R1〜R4 | 1日目の各ラウンド点数（23点以上は赤字） |
| 1日計 | 1日目の合計点 |
| R5〜R8 | 2日目の各ラウンド点数（2日開催の場合のみ） |
| 2日計 | 2日目の合計点（2日開催の場合のみ） |
| 合計 | 全ラウンドの合計点 |
| 平均 | 1ラウンドあたりの平均点 |
| CB | カウントバック値（データがある場合のみ列表示） |
| FR | ファイナルシュートオフ値（データがある場合のみ列表示） |

### CB・FRの凡例

CB・FRの列がある場合、一覧の上部に凡例が表示されます。

- **CB:** カウントバック（小さい数字が上位）
- **FR:** ファイナルシュートオフ（大きい数字が上位）

## フィルター

一覧の上部にフィルターがあります。

- **クラス:** 「全て」またはクラス名（AA / A / B / C）をクリックして絞り込み
- **所属協会:** ドロップダウンから協会を選択して絞り込み

## 統計情報

| 項目 | 説明 |
|------|------|
| 全体平均 | 表示中の有効選手の平均点 |
| 上位6名平均 | 上位6名の平均点 |
| 表示件数 | 現在表示されている選手数 |

## 順位の自動計算

順位は以下のルールで自動計算されます。

1. 合計点の降順
2. 同点の場合はFR（ファイナルシュートオフ）の降順
3. さらに同点の場合はCB（カウントバック）の昇順

失格・棄権の選手は順位なし（最下位扱い）です。

## 成績順位手動設定

自動計算の順位を手動で変更したい場合に使用します。

1. 「成績順位手動設定」ボタンをクリックします
2. モーダル画面が表示されます
3. 選手をドラッグして順位を並び替えます
4. 「保存する」ボタンで確定します

手動設定中は「手動設定中」バッジが表示されます。

「自動計算に戻す」ボタンをクリックすると、手動順位を解除して自動計算に戻ります。

## 閲覧者への公開

成績は自動的に閲覧者ページに反映されます。特別な公開操作は不要です。閲覧者はQRコードまたはURLからアクセスして成績を確認できます。

## 「↺ 更新」ボタン

成績データを最新の状態に更新します。
